# Minions Content-assets

Text posts, image briefs, video briefs, and generated asset references

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-content-assets/sdk`) — Core library for Node.js
- **Python SDK** (`minions-content-assets`) — Python bindings
- **CLI Tool** (`@minions-content-assets/cli`) — Command-line interface
- **Web App** — Interactive playground at [content-assets.minions.wtf](https://content-assets.minions.wtf)
