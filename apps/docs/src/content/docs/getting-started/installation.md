---
title: Installation
description: Install Minions Content-assets packages
---

## TypeScript / Node.js

```bash
npm install @minions-content-assets/sdk minions-sdk
```

## Python

```bash
pip install minions-content-assets
```

## CLI

```bash
npm install -g @minions-content-assets/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
